# Majority Voting System 🗳️

A professional implementation of a 3-input majority voter circuit, developed as part of our Digital Logic Design coursework at the **Faculty of Computers, National University of Sadat City**.

## 🚀 Project Overview
The system is designed to take three binary inputs and output '1' only if the majority (two or more) of the inputs are '1'. This logic is essential in fault-tolerant systems and redundant computing.

## 🛠️ Tech Stack & Tools
* **Logic Simplification:** K-maps & Boolean Algebra.
* **Simulation:** Logisim.
* **Hardware:** Breadboard, Logic Gates (AND, OR), LEDs.
* **Hardware Description Language:** Verilog HDL.

## 📁 Project Structure
* `Majority_Circuit.v`: The core RTL design.
* `Majority_Circuit_tb.v`: Testbench for verification.
* `Presentations/`: Visual slides and circuit diagrams.
